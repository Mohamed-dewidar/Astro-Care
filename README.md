# Astro Care 🚀✨

Astro Care started as a personal project for someone very important to me.

They face ongoing health challenges and need to carefully manage medications, vitamins, and a restricted meal plan every day. What often felt like a complex routine inspired me to build something that could make life a little easier, more organized, and less stressful.

At the same time, I wanted to challenge myself as a software engineer by stepping into a technology I had never used before: React Native.

The app combines health management with a galaxy-inspired user experience. Since the person this app was built for loves space, I designed the interface around a cosmic theme, turning daily health tracking into a more engaging and enjoyable experience.

Development began using Replit's free tier, which helped generate the initial project structure and starter components. From there, I took ownership of the architecture, feature development, UI improvements, and overall product direction.

Throughout the journey, I also leveraged AI tools to accelerate repetitive tasks, learn React Native best practices, explore new ideas, and focus more of my time on solving real problems rather than reinventing boilerplate code.

This project is more than a health-tracking application. It's a blend of care, learning, modern software development, and the belief that technology can make everyday challenges easier for the people we love.

### Tech Stack

* React Native
* Expo
* TypeScript
