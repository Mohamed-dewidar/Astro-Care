

# Build AstroCare – Space-Themed Diet & Medication Companion

Create a production-quality mobile application called **AstroCare**.

AstroCare is a futuristic space-themed health companion designed for people who follow strict dietary schedules and medication routines.

The primary use case is helping users manage multiple meals per day, meal-related medications, reminders, reusable meal plans, and daily diet templates.

The app must feel like a premium futuristic health system running inside a spaceship command center.



# App Name

Tagline:

"Navigate your health mission."

# Smart Medication Logic

Critical Feature:

Medication reminders must be dynamic.

Example:

Breakfast scheduled for 8:00 AM.

Medication is set:

20 minutes after breakfast.

User actually marks breakfast complete at 8:17 AM.

Medication reminder automatically shifts to:

8:37 AM.

This feature is required.

Food

{

{
